# Breaking Bad Memory Card Game

## Live Demo
[(https://fitness-coach-landing.netlify.app/](https://spiffy-nougat-3c091a.netlify.app/)

## Features
- Fully responsive
- Different Levels
- Music effects
- Animations
- Restart/Reset the game

## Tech Stack
- Vite
- React
- CSS

## Installation
- git https://github.com/Felina404/memory-game.git
- cd memory-game
- npm install
- npm run dev

### build
- npm run build
### Preview
- npm run preview
